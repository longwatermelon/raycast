Controls

- WASD: movement
- Mouse motion / arrow keys: rotation
- Left mouse button / space: attack
- Right mouse button: grappling hook
- R: reload
- 1: equip knife
- 2: equip gun

Cheat controls (if applicable)

- c: toggle collision detection
- h: view only horizontal walls
- v: view only vertical walls
- q: view walls normally
- b: add more loaded bullets to gun
- e: xray for entities
- p: add health
- x: let bullets pass through walls
